# Contributing to Verdict

Thank you for your interest in contributing to Verdict!

---

## 1. Setup Instructions

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/saiiexd/Verdict-Agentic-Research-Analyst.git
   cd Verdict-Agentic-Research-Analyst
   ```
2. **Backend Setup**:
   Create a virtual environment and install packages:
   ```bash
   cd backend
   python -m venv venv
   source venv/bin/activate
   pip install -r requirements.txt
   ```
3. **Frontend Setup**:
   Install packages:
   ```bash
   cd ../frontend
   npm install
   ```

---

## 2. Pull Request Process

1. Create a branch for your feature or bug fix:
   ```bash
   git checkout -b feature/my-new-feature
   ```
2. Ensure code is linted and type-checked:
   ```bash
   npm run lint
   ```
3. Submit a pull request detailing your changes.
