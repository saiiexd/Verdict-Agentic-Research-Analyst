# Future Roadmap

Verdict v1.0.0 is fully complete and optimized. This roadmap defines future enhancements planned for subsequent major releases.

---

## 1. Phase F: Enterprise Workspace Sync
- **Backend Database Sync**: Add PostgreSQL/Prisma integrations to move from browser local storage caching to server-side user data synchronization.
- **Collaborative Research**: Real-time collaborative annotations allowing team members to highlight report sentences.

---

## 2. Phase G: Advanced AI Capabilities
- **Multi-Model Debates**: Allow configuring the Critic Agent and Writer Agent to use different LLM providers (e.g. Gemini, OpenAI, Claude) to reduce bias.
- **Streaming Tokens**: Integrate Server-Sent Events (SSE) to stream draft packaging dynamically.
