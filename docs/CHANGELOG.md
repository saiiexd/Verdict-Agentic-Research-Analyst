# Changelog

All notable structural changes to the Verdict project will be documented here.

---

## [1.0.0] - 2026-07-04
### Added
- **Multi-Agent Flow**: LangGraph integrated into FastAPI to orchestrate Financial, News, Writer, Critic, and Refiner agents.
- **Productivity Hotkeys**: `Ctrl+K` Command Palette and sequential keystrokes (`gd`, `gh`, `gs`).
- **Research Panel**: Dynamic report viewer with interactive metric SVG cards and Recharts models.
- **State Storage**: Zustand local persistence slices.
- **Deployment Setups**: Production `.env` definitions, Dockerfile guides, and DevOps pipeline specs.
