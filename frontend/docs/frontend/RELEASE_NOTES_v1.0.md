# Release Notes v1.0.0

Date: July 4, 2026  
Release Version: **v1.0.0-Stable**

---

## 1. Release Highlights

We are proud to announce the stable production release of the Verdict frontend workspace. This launch establishes an AI-powered financial research workspace.

---

## 2. Included Features

- **Command Search and Workspace Shell**: Submission forms for company tickers linked to multi-agent pipelines.
- **AI Workflow Visualization**: Live progress trackers representing financial analysis execution milestones.
- **Premium Report Experience**: Collapsible document views, search query term highlight overlays, and citations.
- **Analytics Dashboards**: Recharts-driven sentiment distributions and valuation comparisons.
- **Workspace Intelligence & History Logs**: Pinning mechanisms, list/grid switches, and sorted query lists.
- **Tab-based Settings Panel**: Configurable themes, reasoning pipeline depths, mock options, and keyboard maps.
- **Command Palette (`Ctrl+K`)**: Fast, keyboard-native command search interface.
