# Accessibility Certification

This document registers the compliance of the Verdict frontend workspace with accessibility best practices.

---

## 1. Compliance Checklist

- [x] **Aria Landmark Roles**: Layout elements are organized using semantic tags (`main`, `aside`, `header`, `nav`, `footer`).
- [x] **Focus Management**: Modal inputs (like the Command Palette) auto-focus upon activation, and focus traps are maintained.
- [x] **Contrast Ratios**: Contrast ratings for all text components meet AAA expectations in both themes.
- [x] **Reduced Motion**: Motion durations are kept short to accommodate users with motion sensitivities.
- [x] **Keyboard Controls**: Tabbing through selects, inputs, and navigation elements follows a logical flow.
